Dr.Egeria Inbox

The Dr.Egeria Inbox is the default location where place markdown files containing Dr.Egeria commands for processing. When you process a Dr.Egeria markdown file it will, by default, place the results in the Dr.Egeria Outbox.

Directory Contents

How to Use

To use Dr.Egeria:

  1. Create a markdown file with Dr.Egeria commands
  2. Place the file in this directory
  3. Process the file by calling `dr_egeria_md` command either from a command line or using the `hey_egeria tui` command line interface./li>
  4. Check the Dr.Egeria Outbox for results
